To set up the build, do the following:

0. Copy eclipse-sdk tar or zip file into test-sandbox/

1. Checkout the mingle mylyn connector projects into a folder/workspace

2. Change the build.properties to point to the right folders

3. Modify the two "version" properties at the top of the build.properties file to match the versions of the plug-ins in your Eclipse installation.

4. When the build completes, navigate to mingle-mylyn-build/I.mingle-mylyn and unzip the com.thoughtworks.mingle.mylyn-mingle-mylyn-linux.gtk.x86_64.zip file to some place on your hard drive.
